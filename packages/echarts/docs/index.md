# ECharts Composables

EChart 是非常好用的可视化图表库。

## Design Principles

可以看到 EChart 提供了非常丰富的 option 选项用于构建可视化图表，但对于实际开发场景来说，基本上都要反复查找 EChart 的 option 配置项文档，每来一个新的图表需求都要重复地编写基本一样的 option 对象。

### Simple / Quick / Default

我们可以使用 `useEChartOption` 方法来快速构造一个马上能用的 echarts option 对象
